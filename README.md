# **Sentinel**
### _A real-time danger detector and community alert system_

**Sentinel** is a crowdsourced danger-detection app that helps users stay safe by showing nearby hazards on a live map.

## **Features**
- **Real-time incident reporting** with forms and geolocation
- **Interactive map:** displaying danger zones and reported incidents using Leaflet
- **Automatic severity classification of incidents** using Gemini API
- **Firebase integration** for authentication and data storage
- **Community voting** to upvote valid alerts or downvote false ones
- **Filter relevant reports** by using the search tool in your area and worldwide

---

## **How It Works**
1. Users submit danger reports (type, description, location).  
2. The system aggregates and filters reports by severity, proximity and reliability.  
3. Other users confirm or dismiss reports through upvotes/downvotes.  
4. The map dynamically updates to reflect trusted, active alerts.

---

## **Tech Stack**
- **Frontend:** React + Leaflet (for interactive maps)  
- **Backend:** Firebase Firestore & Auth
- **APIs:** Geolocation, Leaflet, Gemini
- **UI / Styling:** TailwindCSS for UI, Radix UI for components
